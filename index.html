Про гурток

Наш гурток призначений для учнів, які хочуть вивчити основи програмування на мові Python, навчитися створювати проекти та брати участь у конкурсах з програмування. Ми пропонуємо інтерактивне навчання, яке поєднує теоретичні лекції та практичні завдання.

Приклади завдань, що вирішуються на заняттях

Задача 1: Перевірка простого числа

Напишіть програму, яка перевіряє, чи є введене число простим.

def is_prime(n):
        if n <= 1:
            return False
        for i in range(2, int(n**0.5) + 1):
            if n % i == 0:
                return False
        return True
    
    num = int(input("Введіть число: "))
    if is_prime(num):
        print("Число є простим.")
    else:
        print("Число не є простим.")
